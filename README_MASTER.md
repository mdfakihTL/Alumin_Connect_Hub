# Alumni Portal Backend

This is the master branch. For development work, please use the `dev` branch.

## Branch Structure

- **master**: Production-ready code (stable releases)
- **dev**: Development branch (active development)
- **uat**: User Acceptance Testing branch (pre-production testing)

## Getting Started

Switch to the dev branch for development:

```bash
git checkout dev
```

See the README.md in the dev branch for complete documentation.
